## Learning Path

### Month 1-2: Python & Data Fundamentals
- Python, Pandas, NumPy
- Jupyter, VS Code
- SQL basics

### Month 3-4: Statistics & Visualization
- Statistics, Probability
- Matplotlib, seaborn, Plotly
- EDA techniques

### Month 5-6: ML Basics
- KNN, Naive Bayes, SVM
- Decision Forests, Boosting
- Clustering

### Month 7-8: Time Series & NLP
- ARIMA, Time Series Analysis
- NLTK, Text Mining
- Word2Vec, GloVe

### Month 9-10: Deep Learning Intro
- RNN, LSTM
- Computer Vision basics
- YOLO introduction

### Month 11-12: Integration
- Flask/Django basics
- RESTful APIs
- Project deployment

---

## Progression to Tier 2

Ready for Tier 2 when you can:
- Manipulate data confidently with Pandas
- Build and evaluate basic ML models
- Create visualizations from data
- Use Git for version control
- Write and run SQL queries
- Explain statistical concepts
