## Learning Path

For Beginner level, spread these 53 items across 12 months:

| Month | Focus Areas |
|-------|-------------|
| 1 | Python, Pandas, NumPy, Git, VS Code |
| 2 | Statistics, Probability, Matplotlib, Jupyter |
| 3 | SQL, ETL, Data Science fundamentals |
| 4 | Algorithms: KNN, Naive Bayes, Decision Trees |
| 5 | Time Series, ARIMA, seaborn |
| 6 | Text Mining, NLTK, Word2Vec |
| 7 | Deep Learning intro, RNN, LSTM |
| 8 | Flask, RESTful APIs, Streamlit |
| 9 | Computer Vision intro, YOLO |
| 10 | A/B Testing, Experimental Design |
| 11 | Integration project using multiple skills |
| 12 | Capstone project, portfolio |

For Intermediate and Advanced levels, Tier 1 is foundational and covered in earlier months alongside Tier 2/3.
