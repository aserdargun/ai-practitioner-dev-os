# Technology Tiers

Definition of the three learning tiers and their scope.

---

## Overview

The AI Practitioner Booster 2026 curriculum is organized into three cumulative tiers:

| Level | Tiers Covered | Focus |
|-------|---------------|-------|
| Beginner | Tier 1 | Foundations |
| Intermediate | Tier 1 + Tier 2 | Foundations + Specialization |
| Advanced | Tier 1 + Tier 2 + Tier 3 | Full stack + Production |

---

## Tier 1: Foundations

**Months**: 1-4

**Goal**: Build solid fundamentals in data and ML engineering.

### Topics Covered

| Month | Topic | Key Skills |
|-------|-------|------------|
| 01 | Foundations & Setup | Dev environment, Python, Git, VS Code |
| 02 | Data Engineering | Pandas, data pipelines, validation |
| 03 | ML Fundamentals | Scikit-learn, model training, evaluation |
| 04 | Deep Learning | PyTorch/TensorFlow, neural networks |

### Technologies

**Core**:
- Python 3.11+
- Git & GitHub
- VS Code
- Jupyter notebooks

**Data**:
- pandas
- numpy
- Pydantic
- Great Expectations

**ML**:
- scikit-learn
- PyTorch or TensorFlow
- Weights & Biases (logging)

### Deliverables

By end of Tier 1, you should have:
- [ ] Working development environment
- [ ] Data pipeline with validation
- [ ] Trained ML model with evaluation
- [ ] Deep learning experiment

---

## Tier 2: Specialization

**Months**: 5-8

**Goal**: Develop expertise in AI specialization areas and deployment.

### Topics Covered

| Month | Topic | Key Skills |
|-------|-------|------------|
| 05 | NLP & LLMs | Transformers, RAG, prompt engineering |
| 06 | Computer Vision | CNNs, object detection, image processing |
| 07 | MLOps & Deployment | Docker, FastAPI, model serving |
| 08 | System Design | Architecture, scaling, monitoring |

### Technologies

**NLP/LLM**:
- Hugging Face Transformers
- LangChain
- OpenAI API / Anthropic API
- Vector databases (Chroma, Pinecone)

**Computer Vision**:
- OpenCV
- torchvision
- YOLO / Detection frameworks

**MLOps**:
- Docker
- FastAPI
- MLflow / Weights & Biases
- GitHub Actions

**Infrastructure**:
- AWS/GCP basics
- Kubernetes (concepts)
- Monitoring (Prometheus, Grafana)

### Deliverables

By end of Tier 2, you should have:
- [ ] RAG application with evaluation
- [ ] Computer vision model deployed
- [ ] Containerized ML service
- [ ] System design document

---

## Tier 3: Production & Mastery

**Months**: 9-12

**Goal**: Master production-grade AI systems and build portfolio.

### Topics Covered

| Month | Topic | Key Skills |
|-------|-------|------------|
| 09 | Evaluation & Testing | ML testing, A/B testing, evals |
| 10 | Advanced Topics | Agents, fine-tuning, optimization |
| 11 | Capstone Project | Full-stack AI project |
| 12 | Portfolio & Career | Demo prep, job readiness |

### Technologies

**Evaluation**:
- pytest
- RAGAS
- Custom eval frameworks
- A/B testing tools

**Advanced**:
- LangGraph / Agent frameworks
- Fine-tuning (LoRA, PEFT)
- Optimization (ONNX, quantization)
- Distributed training

**Production**:
- CI/CD pipelines
- Monitoring & alerting
- Cost optimization
- Security best practices

### Deliverables

By end of Tier 3, you should have:
- [ ] Comprehensive test suite
- [ ] Advanced AI application
- [ ] Capstone project (portfolio-ready)
- [ ] Polished portfolio with demos

## Level Expectations

### Beginner (Tier 1 Only)

**Pace**: 10-15 hours/week
**Focus**: Learning fundamentals
**Projects**: Guided with clear steps
**Evaluation**: Completion-focused

### Intermediate (Tier 1 + 2)

**Pace**: 15-20 hours/week
**Focus**: Building skills + deploying
**Projects**: Semi-guided with choice
**Evaluation**: Quality + deployment

### Advanced (Tier 1 + 2 + 3)

**Pace**: 20+ hours/week
**Focus**: Production-ready systems
**Projects**: Self-directed
**Evaluation**: Full rubric (5 dimensions)
