## Learning Path

### Month 1-2: ML Frameworks
- PyTorch or TensorFlow
- scikit-learn advanced
- XGBoost, LightGBM

### Month 3-4: Production APIs
- FastAPI
- Docker
- GitHub Actions
- PostgreSQL

### Month 5-6: LLM Fundamentals
- Hugging Face
- LangChain basics
- Embedding models

### Month 7-8: RAG Systems
- Vector databases (Pinecone, Qdrant)
- RAG architecture
- Evaluation

### Month 9-10: Cloud & MLOps
- AWS/Azure/GCP basics
- MLflow
- Airflow introduction

### Month 11-12: Integration
- Full pipeline project
- Monitoring setup
- Production deployment

---

## Progression to Tier 3

Ready for Tier 3 when you can:
- Deploy containerized ML services
- Build and evaluate RAG systems
- Use cloud platforms confidently
- Set up CI/CD pipelines
- Monitor production systems
