# Tier 2 - Intermediate Shipping

Tier 2 focuses on shipping ML systems to production. This tier is covered by Intermediate and Advanced level learners.

## Overview

**Focus**: Deploy models to production, implement MLOps practices, work with cloud platforms, and build advanced ML systems.

**Duration**: Combined with Tier 1 in 12 months for Intermediate level

**Goal**: Ship production-ready ML systems with proper monitoring and CI/CD.

## Prerequisites

Before starting Tier 2, you should have:
- Completed Tier 1 fundamentals
- Solid Python and SQL skills
- Basic ML model building experience
- Familiarity with Git and command line

## Learning Objectives

By the end of Tier 2, you should be able to:

1. **Deploy to Production**
   - Containerize applications with Docker
   - Deploy to cloud platforms
   - Set up CI/CD pipelines

2. **Implement MLOps**
   - Track experiments with MLflow
   - Version models and data
   - Automate model training

3. **Build Advanced ML**
   - Fine-tune LLMs
   - Implement RAG systems
   - Use vector databases

4. **Monitor Systems**
   - Set up metrics and alerts
   - Create dashboards
   - Debug production issues

5. **Work with Cloud**
   - Use managed ML services
   - Implement serverless functions
   - Manage cloud resources

## Next Steps

For Intermediate learners, Tier 2 is combined with Tier 1 accelerated:
1. Master Tier 1 fundamentals quickly
2. Start deploying to production
3. Build increasingly complex systems
