## Learning Path

For Intermediate level, Tier 2 is covered alongside Tier 1 across 12 months:

| Month | Tier 1 Focus | Tier 2 Focus |
|-------|--------------|--------------|
| 1-2 | Foundation | Docker, CI/CD basics |
| 3-4 | ML algorithms | scikit-learn, XGBoost |
| 5-6 | Data processing | PostgreSQL, Redis, FastAPI |
| 7-8 | NLP basics | LangChain, RAG, Vector DBs |
| 9-10 | Cloud basics | AWS/Azure/GCP services |
| 11-12 | Integration | MLflow, monitoring, deployment |

For Advanced level, Tier 2 is integrated with Tier 3 throughout the year.
