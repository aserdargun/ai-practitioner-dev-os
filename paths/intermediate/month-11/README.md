**Focus**: Build agentic systems with LangGraph and tool use.

---

## Why It Matters

AI agents are the next frontier of LLM applications. Understanding how to build systems that can reason, use tools, and accomplish complex tasks is essential for cutting-edge AI development. This month prepares you for the future of AI.

**Job Relevance**: Emerging high-value skill; early expertise is valuable.

## Main Project: Agentic Assistant

Build an AI assistant that can use tools and accomplish multi-step tasks.

### Deliverables

1. **Agent System** with multiple tools
2. **Tool Implementations** for useful capabilities
3. **State Management** for multi-turn interactions
4. **Evaluation Suite** for agent behavior
5. **GitHub Repository** with documentation

### Definition of Done

- [ ] LangGraph agent architecture defined
- [ ] At least 5 tools implemented
- [ ] Agent can chain tool calls
- [ ] State persists across interactions
- [ ] Error handling for tool failures
- [ ] Human-in-the-loop for sensitive actions
- [ ] Evaluation suite for agent tasks
- [ ] Edge cases handled
- [ ] Documentation complete
- [ ] Demo shows complex task completion
