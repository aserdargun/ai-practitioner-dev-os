# Month 12: Capstone Project

**Focus**: Build a complete portfolio project showcasing your full skill set.

---

## Why It Matters

The capstone is your opportunity to integrate everything you've learned into a polished, portfolio-ready project. This is what you'll show employers, use in interviews, and point to as evidence of your capabilities.

**Job Relevance**: The most important month; your capstone can define your job search success.

---

## Prerequisites

- Month 01-11 complete
- All previous projects available
- Clear understanding of your target roles

---

## Learning Goals

**Integration of All Tiers**:
- Everything from Tier 1 + Tier 2
- Choose technologies that align with your goals
- Demonstrate depth and breadth

**Skills**:
- System design
- Full-stack implementation
- Production deployment
- Technical communication

---

## Main Project: Portfolio Capstone

Build a substantial project that demonstrates your capabilities as an AI practitioner.

### Project Ideas

| Idea | Technologies | Complexity |
|------|--------------|------------|
| AI-Powered Analytics Dashboard | RAG, FastAPI, React, Cloud | High |
| MLOps Platform for Teams | MLflow, Airflow, Docker, K8s | High |
| Conversational Data Assistant | LangGraph, RAG, Streamlit | Medium-High |
| Real-time ML System | Streaming, ML, Monitoring | High |
| Custom LLM Application | Fine-tuning, Evaluation, API | Medium-High |

### Deliverables

1. **Working System** deployed and accessible
2. **Technical Documentation** explaining architecture
3. **Demo Video** (5-10 minutes)
4. **Blog Post** (2000+ words)
5. **GitHub Repository** showcasing best practices

### Definition of Done

- [ ] System is deployed and working
- [ ] Architecture documented with diagrams
- [ ] README is comprehensive
- [ ] Tests cover critical paths
- [ ] CI/CD pipeline works
- [ ] Monitoring is in place
- [ ] Demo video recorded
- [ ] Blog post written
- [ ] LinkedIn post prepared
- [ ] Ready for portfolio presentation

---

## Weekly Cadence

### Week 1: Design & Plan
- Select project scope
- Design architecture
- Set up repository
- Create project plan

### Week 2: Core Implementation
- Build core functionality
- Set up infrastructure
- Implement key features

### Week 3: Complete & Deploy
- Finish implementation
- Deploy to production
- Add monitoring
- Write tests

### Week 4: Polish & Ship
- Create documentation
- Record demo
- Write blog post
- Final presentation
