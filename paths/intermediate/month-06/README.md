# Month 06: LLM Applications

**Focus**: Build applications with Large Language Models using LangChain.

---

## Why It Matters

LLMs are transforming how we build software. Understanding how to work with APIs, chains, and prompts is essential for building modern AI applications. This month introduces the patterns you'll use extensively in RAG and agent development.

**Job Relevance**: Extremely high demand; core skill for AI/ML engineers in 2026.

---

## Prerequisites

- Month 01-05 complete
- Understanding of transformers
- API usage experience

---

## Learning Goals

**Tier 2 Technologies**:
- LangChain (primary focus)
- GPT/Claude API usage
- Prompt engineering
- Chains and memory
- Embedding models

**Skills**:
- Prompt design
- Chain composition
- Context management
- Basic RAG concepts

---

## Main Project: Conversational Chatbot

Build a chatbot that can have conversations, remember context, and answer questions.

### Deliverables

1. **Chatbot Application** with conversation memory
2. **Prompt Templates** for different use cases
3. **Chain Implementations** for complex workflows
4. **Evaluation Suite** for response quality
5. **GitHub Repository** with documentation

### Definition of Done

- [ ] LLM API integration working
- [ ] Conversation memory implemented
- [ ] Multiple prompt templates created
- [ ] At least one complex chain built
- [ ] Basic evaluation of responses
- [ ] Streamlit/Gradio interface
- [ ] Error handling for API failures
- [ ] Cost tracking implemented
- [ ] Documentation complete
- [ ] Demo ready

---

## Weekly Cadence

### Week 1: LLM Basics

- Build prompt templates
- Create chains
- Add conversation memory

### Week 3: Complex Workflows
- Multi-step chains
- Conditional logic
- Error handling
- Basic evaluation

### Week 4: Polish & Ship
- Add UI (Streamlit)
- Finalize evaluation
- Demo and write-up
- Retrospective
