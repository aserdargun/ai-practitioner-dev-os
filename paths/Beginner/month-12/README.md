## Portfolio Checklist

For each project, ensure:

### Code Quality
- [ ] Code runs from fresh clone
- [ ] Dependencies documented
- [ ] No hardcoded paths/secrets
- [ ] Follows style guide
- [ ] Tests exist and pass

### Documentation
- [ ] README explains what, why, how
- [ ] Setup instructions work
- [ ] Results are shown
- [ ] Limitations acknowledged

### Presentation
- [ ] Demo available (live/video/screenshots)
- [ ] 2-3 sentence summary
- [ ] Technical highlights noted
- [ ] Impact/results shown
