## Objectives

By the end of this month, you will:

1. Understand transformer architecture
2. Work with LLMs effectively
3. Build RAG applications
4. Evaluate NLP/LLM systems

---

## Weekly Breakdown

### Week 1: NLP Fundamentals

**Focus**: Text processing basics

**Tasks**:
- [ ] Learn text preprocessing
- [ ] Understand tokenization (BPE, WordPiece)
- [ ] Explore embeddings (Word2Vec, BERT)
- [ ] Use Hugging Face transformers library

**Deliverables**:
- Text processing pipeline
- Embedding exploration notebook

### Week 2: Working with LLMs

**Focus**: Practical LLM usage

**Tasks**:
- [ ] Set up OpenAI/Anthropic API access
- [ ] Learn prompt engineering techniques
- [ ] Implement structured outputs
- [ ] Handle rate limits and errors

**Deliverables**:
- LLM interaction utilities
- Prompt templates library

### Week 3: Building RAG Systems

**Focus**: Retrieval-Augmented Generation

**Tasks**:
- [ ] Set up vector database (Chroma)
- [ ] Implement document chunking
- [ ] Build retrieval pipeline
- [ ] Create RAG chain

**Deliverables**:
- Working RAG system
- Document Q&A application

### Week 4: LLM Evaluation

**Focus**: Evaluate LLM applications

**Tasks**:
- [ ] Create evaluation dataset
- [ ] Implement RAGAS metrics
- [ ] Build evaluation harness
- [ ] Analyze and improve

**Deliverables**:
- Evaluation framework
- Improvement report
