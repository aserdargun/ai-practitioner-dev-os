## Capstone Options

Choose ONE project that showcases your best work:

### Option A: Full-Stack ML Application
Build an end-to-end ML application:
- Data pipeline (Month 2)
- ML model (Months 5-6)
- API backend (Month 10)
- Dashboard frontend (Month 11)

**Example**: Customer churn prediction system with dashboard for sales team.

### Option B: Time Series Forecasting System
Build a complete forecasting solution:
- Data ingestion and cleaning
- Forecasting models (Month 7)
- Interactive dashboard
- Automated updates

**Example**: Sales forecasting dashboard for retail planning.

### Option C: NLP Application
Build a text analytics application:
- Text processing pipeline (Month 8)
- Classification or analysis
- Web interface
- Real-time predictions

**Example**: Customer feedback sentiment analyzer with trends dashboard.

### Option D: Your Own Idea
Propose your own project that:
- Uses 3+ skills from the year
- Solves a real problem
- Can be completed in 4 weeks
- Is demo-able and deployable

## Deliverables

### Code Repository
- Clean, documented code
- Tests with >70% coverage
- README with setup instructions

### Deployment
- Publicly accessible demo

### Documentation
- Project description (why, what, how)
- Architecture diagram

- API documentation (if API)

### Presentation
- 5-minute demo video

## How to Publish

### Demo Video Structure
1. (30s) Introduce the problem
2. (30s) Show the solution overview
3. (2min) Walk through key features
4. (1min) Explain technical highlights
5. (1min) Share what you learned

### Blog Post Structure
1. The Problem
2. The Solution
3. How It Works (with screenshots)
4. Technical Deep Dive (code snippets)
5. What I Learned
6. Try It Yourself (links)

### Portfolio Placement
