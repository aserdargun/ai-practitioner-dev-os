# Learning Journal

Your weekly and monthly reflections live here.

## Purpose

The journal helps you:
- Track what you learn
- Identify patterns
- Celebrate progress
- Build reflection habits

## Templates

- [weekly-template.md](weekly-template.md) — Copy for each week
- [monthly-template.md](monthly-template.md) — Copy for each month

## Naming Convention

### Weekly Files
```
month-01-week-1.md
month-01-week-2.md
month-02-week-1.md
...
```

### Monthly Files
```
month-01-summary.md
month-02-summary.md
...
```

## Creating Entries

### Via Hook
```bash
./.claude/hooks/pre_week_start.sh
```
This creates the week file automatically.

### Manually
1. Copy the appropriate template
2. Rename with month and week number
3. Fill in as you go

## What to Write

### Daily
- What you worked on
- What you completed
- Any blockers

### Weekly
- Goals achieved
- What was challenging
- What you learned
- Best practices to capture

### Monthly
- Overall progress
- Skills developed
- Projects completed
- Goals for next month

## Tips

1. **Write little, often** — Quick notes during the day beat long entries at week end
2. **Be honest** — This is for you, not for show
3. **Focus on learning** — Not just what you did, but what you learned
4. **Review periodically** — Look back to see how far you've come
